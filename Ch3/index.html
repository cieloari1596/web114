JavaScript Exercises

Questions

  1. Question 1: Name a situation where you would need to create an "if" statement.
    An "if" statement is used when you want to execute a certain block of code only if a specific condition is true. For example, you might use an "if" statement to check if a user is logged in before allowing them to access certain pages on a website.
  2. Question 2: When comparing values, in what data type is the result stored?
    When comparing values, the result is stored as a boolean data type, either true or false.
